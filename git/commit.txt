Сначала открываем файл git в своём локальном репозитории на компьютере. Потом в файле текстовом вводим новые символы. Сохраняем. Добавляем изменённый файл - git add . Потом создаем commit (называем как угодно его). Дальше пушим на Git Hub (git push)
